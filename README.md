## ¿Què es TypeScript ?
Lenguaje de programación
Superset de JavaScript
Para añadir tipos al lenguaje

## ¿Por qué usar TypeScript?
1-Seguridad
2-Mantenibilidad
3-Refactorización

Para evitar uno de los problemas de JavaScript es que es un lenguaje con tipado dinámico.

Ejemplo:
2 + '2'='22'

## ¿Qué no hace TypeScript?
Analiza en tiempo de compilación, de forma estática.
No arregla los problemas en tiempo de ejecución.
